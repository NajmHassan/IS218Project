Vi

Vi is a built-in text editor for the Linux operating system originally designed for the Unix operating system. It allows for editing and text documents within the terminal. To access vi, you must use the command vi followed by the name of file you would like to edit. If there is no existing file, then you must create one first using the touch command followed by the name of the file. This will create a file with the designated name and then allow it to be edited.

An example using vi would be:

touch test.txt

vi test.txt

After entering the vi text editor press i to enter insert mode. With this you can edit the file and add text to it. When finished editing press esc. From here you can either quit out of the vi editor, write to the file (save changes), or both. To quit without saving use the command :q. To write to the file use command :w. To write and then quit use the command :wq.

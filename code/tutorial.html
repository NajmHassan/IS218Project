Website tutorial

to Start this website you will need all of the components we talked about in this demo website starting with making a new folder in the terminal

mkdir websiteTutorial creates a new directory

cd websiteTutorial go inside the new directory

touch Dockerfile create a new docker file

touch docker-compose.yml creates a new docker compose file

mkdir public_html(create a new dir where we will store our HTML files in

vi Dockerfile edit Dockerfile (click i to insert or edit once you enter the file). Now we will populate the file with the following code:

FROM php:5.6-apache-jessie
RUN sed -i '/jessie-updates/d' /etc/apt/sources.list
RUN apt-get update
RUN apt-get update --fix-missing
RUN apt-get install git cron rcconf libzip-dev zlib1g-dev libpng-dev libxml2-dev -y
RUN a2enmod rewrite
WORKDIR /var/www/html
COPY ./ .
RUN service apache2 restart
EXPOSE 8000

vi docker-compose.yml edit docker-compose.yml (click i to insert or edit once you enter the file). Now we will populate the file with the following code:

version: '3.5'
services:
'username':
build: .
volumes:
- ./:/var/www/html
ports:
- 3001:80
image: 'dockerhubID'/demo:latest

now we can create our HTML files. so lets run the following command to enter our directory cd public_html and then create a new file with the comman touch index.html then vi index.html edit the file (click i to insert or edit once you enter the file). Now we will populate the file with a basic header that says 'website tutorial'

now that we have everything set, lets use docker (if not installed click here to install) to build this locally to make sure it works

docker-compose up --build and wait till it is done builing. when it is done, go to your browser paste this link: http://localhost:3001/. you should see the html file you wrote!

now we can push our code into dockerhub to be able to pull it from azure Cloud so lets run docker-compose push

now that we have our project on dockerhub. lets create an azurecloud account to run our project globally.

click here to create an Azure account. Once you are done, get back on the terminal and run docker login azure which will take you to your browser to login.

After logging in, youll want to create a new context to be able to use Azure cloud. run docker context create aci AzureCloud and then docker context use azureCloud

After switching context, start running your project by running the following command docker run 'something' 'IPADDRESS' -dp 80:80 'dockerhubID'/demo:latest

when it is done running, copy the 'IPaddress' specified and paste it in your browser! youll see the file you have running on Azure which means anyone can visit it with the specifed IPADDRESS!

Pushing to github repo

when you are ready to push your code to github, go ahead and check the git tab. start by going to the root file. then write these list of commands:

git init

git add .

git commit -m "your message"

git push remote (branch name)

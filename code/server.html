Apache

Apache is a web server that is responisble for handling and acccepting HTTP requests from internent users and then sends them to their desired location.The job of apache is to host our website and allow users to see it once they click on the link.

Installing Apache

To install apache through Ubuntu use this link Ubuntu Download

Once downloaded you can download Apache through Ubunut's repository. Use the command $sudo apt update then the command $sudo apt install apache2. This will update the local package and then install the apache package. To check the app profiles use the command $sudo ufw app list . To switch to the profile ise the command $sudo ufw allow 'Apache'. This will swap to the Apache profile. To check the web server use the command sudo systemctl status apache 2

Home Page

This tutorial will teach the user how to use terminal commands to create directories and files, edit them, and then push them to docker and then azure cloud. The technologies that will be discussed are git, vi, docker, and Nginx.

Getting Started

The first command and you can use is pwd this will display the pathway/directory you are in. The ls command will show you the contents of the directory. To create a directory, use the mkdir command and follow it with the name of the directory. Example: mkdir testDirectory This will create a directory named testDirectory. Then use the cd command to change directories. Be sure the command is followed by the directory name you would like to switch to. Example: cd testDirectory After this use the pwd and ls commands to make sure that you are in the right place and the files look correct. If it is a new directory there will be no files listed. Now that the directory is created it is time to add files and edit them which will be in the next section titled Vi.

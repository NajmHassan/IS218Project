Git

Git is the most widely used modern version control system. Git allows a user to enter commands into the shell do things such as create files, edit files, along with completing commits, pull requests, and push documents, and more. There are many different commands that git can do.

Git Commands:

git init – initializes

git add (filename) – adds file to git

git commit -m “message” – commits the file

git checkout -b (branch name) – creates a branch and switches to it

git push remote (branch name) – pushes the commit to the branch

git status – shows the files to be committed or if there have been commits

After committing the files and pushing them to GitHub the files should be in your github repository
